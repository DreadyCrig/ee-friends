The Friends:Group_Wall_Form function allows your users to submit simple comments to Friends groups, similar to the way Facebook's Group Comment Wall used to be in the earlier days. Only users that belong to the group can post a comment on the group wall. Used in conjunction with the Friends:Group_Wall tag.

{exp:friends:group_wall_form} content {/exp:friends:group_wall_form}
  • Parameters
  • Form Fields
  • Examples

Parameters

The following parameters are available for use:

  • friends_group_id
  • friends_group_name
  • return

friends_group_id=

friends_group_id="{segment_3}"

This parameter is required to feed the group ID of the group wall which you wish to post a comment to. You can hardcode a group ID, pass it through an embed, or grab it from the URI.

friends_group_name=

friends_group_name="{segment_3}"

This parameter is required to feed the group name of the group wall which you wish to post a comment to. You can hardcode a group name, pass it through an embed, or grab it from the URI.

return=

return="friends/group/{segment_3}"

This parameter allows you to return the user to a specified template upon successful submittal of form. If you leave this parameter out, the current page will just reload.

Form Fields

The following form fields are available for use:

  • friends_wall_comment

friends_wall_comment

<textarea name="friends_wall_comment"></textarea>

This form field is where the user would enter in their comment to post to a given group wall.

Examples

The following example will allow your users to submit a wall comment to a given group wall:

<h3>Post a Comment:</h3>

{exp:friends:group_wall_form
    friend_group_id="{segment_3}"
}
    <p><textarea name="friends_wall_comment"></textarea></p>
    <p><input type="submit" name="submit" value="Comment" /></p>

{/exp:friends:group_wall_form}
